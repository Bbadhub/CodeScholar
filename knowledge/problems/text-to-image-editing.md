# Problem: Text-to-Image Editing

Text-to-image editing involves modifying images based on textual descriptions or commands. This allows users to make changes to images without needing advanced graphic design skills.

## You Have This Problem If

- You want to allow users to edit images using text prompts.
- You need a solution that is accessible to non-professionals.
- You require quick edits without heavy computational demands.
- You are integrating image editing features into an application.
- You are looking for a user-friendly interface for image modifications.

## Start Here

**Start with LightEdit for most cases, as it provides a balance of speed and ease of use, making it ideal for non-professionals.**
